# 🏠 House Price Prediction

A Machine Learning project that predicts house prices based on area and number of bedrooms using Linear Regression. This project demonstrates the fundamentals of predictive analytics and regression modeling using Scikit-Learn.

---

## 📌 Features

- Predicts house prices using machine learning
- Uses Linear Regression algorithm
- Data preprocessing with Pandas
- Simple and easy-to-understand implementation
- User input-based predictions

---

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Machine Learning
- Linear Regression

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/house-price-prediction.git
```

### Move to Project Folder

```bash
cd house-price-prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python house_price_prediction.py
```
